# QMK workspace metadata

This repository contains the workspace-level metadata for my QMK environment. It is intentionally separate from the actual code repositories and only tracks root-level instructions, Codex context, helper scripts, and similar workspace files.

The firmware and userspace repositories are cloned separately into this workspace as `./qmk_firmware/` and `./qmk_userspace/`. They are managed as independent Git repositories with their own remotes and history.

The canonical Codex skill for this workspace is located at `./.agents/skills/qmk-workflow/`.
